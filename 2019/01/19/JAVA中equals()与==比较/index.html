equals()与==比较

发表于 2019-01-19 阅读次数：
本文字数： 490 阅读时长 ≈ 1 分钟

==比较的是对象的引用地址。

如果不重写equals()方法，则默认比较的是对象的引用地址，因为Object类中的equals()方法比较的是对象的引用地址

特别的，String类已经重写了equals()方法
