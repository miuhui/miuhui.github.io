Array与ArrayList

发表于 2019-03-18 阅读次数：
本文字数： 211 阅读时长 ≈ 1 分钟

区别

  • 实现。Array是一个数据结构，只暴露了length方法；ArrayList是JAVA中的一个集合类，有更多可以调用的方法。
  • 迭代。ArrayList与Array都可以通过for、while等循环遍历元素，但是ArrayList还提供了Iterator和ListIterator迭代器的方式进行遍历。
  • 扩容。Array大小固定，ArrayList可以动态改变。

相同点

  • O(1)获取一个元素
  • 都容许存储Null值
  • 都容许存储重复值
