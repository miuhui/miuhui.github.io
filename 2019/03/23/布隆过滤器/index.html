布隆过滤器

发表于 2019-03-23 阅读次数：
本文字数： 195 阅读时长 ≈ 1 分钟

组成

位图 + 一系列映射函数

为什么要多个函数？减少hash冲突

特征

  • 布隆过滤器判断数据不存在，那么一定不存在
  • 布隆过滤器判断数据存在，数据可能不存在

实现

Guava中封装了布隆过滤器的实现

布隆过滤器的误判率默认是3%，也可以根据业务需要设置误判率，误判率越低位图越大、映射函数越多。

扩展

默认实现的Bloom Filter不提供删除功能，可以使用CountingBloomFilter，即加入一个计数器。
